¿Que es composer?

Es una herramienta para la gestión de dependencias en PHP para nuestros proyectos, facilitando su administración (instalación y actualización).

INSTALACIÓN LINUX

Antes de instalar

Necesitamos instalar las siguientes dependencias antes de instalar composer.-


$ sudo apt-get install curl php5-cli git

Nota: Si utilizas PHP 7, cambia "php5-cli" por "php7.0-cli".

Instalación composer

Descargamos e instalamos composer como un comando del sistema.-


$ curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin --filename=composer

Verificamos la instalación

Para comprobar que se instalo correctamente ejecutamos el siguiente comando.-


$ composer

Debemos ver algo parecido

CREAR COMPOSER.JSON

¿Para que nos sirve?

Nos ayuda a gestionar las versiones de las dependencias administradas por composer en un proyecto, de esta forma el proyecto estará consistente en todo momento.

Crear el archivo composer.json

Debemos crear un archivo llamado "composer.json" en el directorio raíz de nuestro proyecto, puede seguir el siguiente ejemplo.-


{
    "require": {
    }
}

INSTALACIÓN DE DEPENDENCIAS

Instalar dependencias

Puede instalar dependencias con composer utilizando el siguiente comando.-.


$ composer require nombre_dependencia

Archivo composer.lock

Después de instalar la dependencia, composer almacena la versión instalada en el archivo composer.lock, de esta forma, el proyecto se mantendrá consistente trabajando siempre con las mismas versiones.

Directorio de instalación

El directorio por defecto es "vendor" ubicado en la raíz de nuestro proyecto.-


+ Directorio raiz
    - vendor
    composer.json
    composer.lock

Composer y git

Si utilizas git como sistema de subversiones, es recomendable añadir el directorio "vendor" al archivo ".gitignore" para obligar a instalar las dependencias cada ves que se descargue el proyecto.

Por ultimo no olvides subir los archivos composer.json y composer.lock.

AUTOLOAD

¿Para que nos sirve?

Se trata de un sistema de autocargado para nuestras dependencias, es decir con un solo "require" podemos añadir todas las dependencias a nuestro archivo PHP para trabajar

Utilizar el autoload

Añade la siguiente linea al archivo PHP en el que vas a incluir las dependencias.-


require_once "vendor/autoload.php"

// Aqui inicializamos las dependencias

Nota.- La inicialización depende de cada dependencia.
