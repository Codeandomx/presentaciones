COMENCEMOS

JCheckBox

Los cuadros de verificación (JCheckBox) se utilizan para seleccionar varias opciones al mismo tiempo.

Métodos

isSelected()

Permite saber si el cuadro de verificación esta seleccionado retornando un valor true, o false en caso contrario.


if(checkBox.isSelected()){
	// Ejecutamos si esta seleccionado
} else {
	// Ejecutamos si no esta seleccionado	
}
